SECURE CLOUD APPLICATION

Security is one of the major issues in Cloud. To handle it, the client application encrypts data and send it to the server, the cloud serve, breaks the files into different parts using erasure modelling so as to prevent data loss, data can be recovered using the parts even if some are lost.
